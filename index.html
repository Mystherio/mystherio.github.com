21/06/16 - Partage de Connexion (Windows) :

I - Création du logiciel
1) Ouvrez le boc note et entrez collez ceci a l'interieur:
:menu
cls
@echo off
title Partage de Connexion
echo 1 : lancer
echo 2 : arreter
echo 3 : modifier
echo 4 : centre reseau et partage
echo 5 : infos
choice /C:12345
if %ERRORLEVEL% == 1 goto lancer
if %ERRORLEVEL% == 2 goto arreter
if %ERRORLEVEL% == 3 goto modifier
if %ERRORLEVEL% == 4 goto centre
if %ERRORLEVEL% == 5 goto info
:modifier
cls
set /p saisie1=Nom du Reseau:
set /p saisie2=Mot de Passe:
netsh wlan set hostednetwork mode=allow ssid=%saisie1% key=%saisie2%
pause
goto menu
:lancer
cls
netsh wlan set hostednetwork mode=allow
netsh wlan start hostednetwork
pause
cls
goto menu
:arreter
cls
Netsh wlan stop hostednetwork
Netsh wlan set hostednetwork mode=disallow
pause
goto menu
:centre
control.exe /name Microsoft.NetworkAndSharingCenter
cls
pause
goto menu
:info
cls
netsh wlan show hostednetwork
pause
goto menu
2) Allez dans fichier > enregistrer sous...
Dans Type, selectionnez "tous les fichiers".
Mettez connexion.bat en nom de fichier.

II - Parametrer le logiciel
Allez tout d'abord dans Modifier (3)
Entrez un nom de reseau, sans espace ou caracteres speciaux.
Entrez un mot de passe compose de 8 caractères minimum (uniquement lettres et chiffres)

Allez dans Lancer (1)

Allez dans Centre reseau et partage (4)
Cliquez sur votre connexion (ex: Wifi (livebox...))
Allez dans l'onglet Partage
Cochez la case "Autoriser d'autres utilisateurs à se connecter via la connexion internet de cet ordinateur"
Selectionnez  "connexion réseau local" comme réseau domestique.
Cochez "autorisez d'autres utilisateurs a controler ou désactiver la connexion internet partagee".

Pour l'areter il vous suffira d'aller dans le logiciel de d'aller dans Arreter (2)
